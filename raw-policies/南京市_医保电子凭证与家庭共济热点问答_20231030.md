# 12393热点直通（第6期）——儿童医保电子凭证与家庭共济

发布时间：2023-10-30  
来源：南京市医疗保障局

## 一、儿童医保电子凭证如何激活、使用

儿童的医保电子凭证需通过绑定家长的医保电子凭证亲情账户进行激活。

1. 家长在“国家医保服务平台”APP、支付宝等渠道激活自己的医保电子凭证。
2. 在亲情账户中添加儿童医保电子凭证，真实、准确、完整地填写被绑定人基本信息，上传相关材料。
3. 儿童年龄 ≤16 周岁的，需上传出生证明或双方户口本等资料，审核通过后即绑定成功。
4. 绑定成功后，即可使用儿童的医保电子凭证进行医保支付、查询等。

## 二、家庭共济账户如何开通、使用

根据家庭成员医疗需求，按照自愿原则，可通过以下渠道申请设立家庭共济关系：

### 共济双方均为南京参保

- “南京医保”微信公众号 → 服务大厅 → 医保家庭账户管理
- “我的南京”APP → 医保服务 → 医保家庭账户管理

### 共济双方既有南京参保，也有本省其他城市参保

- “江苏医保云”APP → 我要办 → 家庭共济

### 使用规则

设立家庭共济账户关系后，共济对象在医院就医使用本人的医保卡刷卡结算时，即可使用家庭共济账户余额。
