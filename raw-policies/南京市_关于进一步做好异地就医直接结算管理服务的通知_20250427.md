# 关于进一步做好异地就医直接结算管理服务的通知

文号：宁医发〔2025〕5号  
发布单位：南京市医疗保障局  
发布时间：2025年4月27日

## 一、备案类型与办理方式

### （一）异地长期居住人员

适用于异地长期居住、工作或学习人员。

**1. 正常备案**

申请备案时提交（上传）相关异地证明材料：

- 异地居住：异地户口本、房产证、居住证或其他有效异地居住证明（任选其一）。
- 异地学习与工作：学习证、培训证明或其他能够证明异地学习工作的材料（任选其一）。

**2. 承诺备案**

因暂未获取异地证明材料等原因无法通过“正常备案”申请的，可先行进行承诺备案。备案后需要在30日内通过原申请渠道补齐材料，否则该条异地长期备案信息将予以自动注销。补传材料的要求同“正常备案”。

注意事项：承诺备案开始至补齐材料时间内不得变更备案信息；承诺备案到期后未履行承诺补齐材料的，不得再次申请承诺备案；备案后6个月内不可注销。

### （二）临时赴异地人员

**1. 转外就医**

申请备案同时，需要提交（上传）由南京三级医院主任或副主任医生签字、医院医保部门审批确认盖章的《转外就医备案表》。

**2. 其他临时外出**

无需提交材料，登记备案信息即可。

## 二、备案渠道

- “南京医保”微信公众号 → 服务大厅 → 异地就医备案。
- “我的南京”APP → 医保服务 → 异地就医备案。
- “江苏医保云”APP。
- “国家医保服务平台”APP / “国家异地就医备案”微信小程序。
- 南京市各级医保经办机构窗口。

## 三、备案有效期

| 备案类型 | 有效期 |
|---|---|
| 异地长期居住 | 与材料有效期保持一致；从备案开始时间起6个月内不允许注销 |
| 转外就医 | 从备案开始时间起1年内有效 |
| 其他临时赴异地 | 从备案开始时间起6个月内有效，可随时申请注销 |

## 四、结算待遇

参保人员省内联网结算执行参保地（南京）医保目录和报销政策；跨省联网结算执行就医地医保目录、参保地（南京）医保报销政策。

| 异地就医类型 | 刷卡结算待遇 | 零星报销待遇 |
|---|---|---|
| 异地长期居住 | 不降低 | 不降低 |
| 异地长期居住急抢救人员 | 不降低 | 不降低 |
| 转外就医 | 降低10个百分点 | 降低10个百分点 |
| 转外就医急抢救人员 | 降低10个百分点 | 降低10个百分点 |
| 其他临时赴异地 | 降低20个百分点 | 降低20个百分点 |
| 其他临时赴异地急抢救人员 | 降低10个百分点 | 降低10个百分点 |
| 未备案就医人员手工报销 | — | 降低20个百分点 |

## 五、取消备案

- 长期异地居住人员备案6个月内不得取消备案。
- 异地转诊人员备案和其他临时外出就医人员备案可随时取消备案。

取消渠道：南京市各级医保经办柜台、移动端（“我的南京”APP、“江苏医保云”APP、“南京医保”微信公众号）。

## 六、多地就医

参保人员最多可办理3条异地就医备案，其中最多只有1条备案可按“承诺备案”方式办理，其他2条备案需按正常备案流程办理。办理异地就医备案后，南京本地就医结算不受影响。
