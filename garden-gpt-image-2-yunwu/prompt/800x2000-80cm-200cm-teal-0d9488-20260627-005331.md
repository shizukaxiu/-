竖版 800x2000“医保经办助手”产品海报，宽 80cm、高 200cm。

明亮干净、专业可信的医保服务风格，白色/浅灰背景，teal 青绿色 (#0d9488) 主高亮，圆角卡片、柔和阴影，无深色科技感。

构图有呼吸感、非对称、突出双端能力：
- 顶部：大号标题“医保经办助手”，副标题“让老百姓像聊微信一样办医保”，左右各一个角色徽章“普通用户端”和“系统管理员端”。
- 左上（用户端）：手机对话界面，展示文字咨询、语音输入、OCR 发票识别、数字人头像；右侧浮出 PolicyCard 政策来源卡片。
- 右上（管理员端）：管理员后台界面，展示 PDF/DOC 文件上传区、知识库文件列表、重建进度条、RAG 索引状态。
- 视觉重心：RAG 知识库构建流程图——raw-policies 政策文档 → 文档解析 → 400 字符切片 → TF-IDF 向量化 → LanceDB + JSON 双索引 → /api/search 检索 → 对话中展示 PolicyCard。
- 中下部：附近定点机构地图网络，医院/药店/服务站节点连线，体现关系网络。
- 底部流程：多模态输入 → 意图理解 → RAG 检索 → 大模型回答/表单引导 → 办事闭环（异地备案、发票报销、账户看板、机构推荐）。
- 最底部：技术栈标签 React 19 + Vite + TypeScript + TailwindCSS v4、DeepSeek、TF-IDF/LanceDB、Tesseract OCR、Web Speech API，以及口号“随时随地 · 想问就问 · 一站办结”。
